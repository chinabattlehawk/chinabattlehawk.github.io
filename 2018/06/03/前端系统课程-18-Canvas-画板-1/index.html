前端系统课程 18. Canvas 画板 (1)

鼠标事件

  • 鼠标按下 onmousedown
  • 鼠标移动 onmousemove
  • 鼠标松开 onmouseup

鼠标位置

screenX/Y 代表鼠标在整个屏幕的位置；clientX/Y 代表鼠标在窗口或文档的位置；

canvas 标签

由于使用普通元素无法在短的时间内连贯的监听鼠标位置，导致“点”的连接不连贯，所以用普通元素做画板很不理想。

  • <canvas> 标签和 <img> 标签类似，自带宽和高属性；
  • 如果用 CSS 设置 Canvas 元素的宽高可能会导致图像扭曲，所以最好用它自带的宽高属性设置；
  • 使用 Canvas 之前需要先设置上下文，例如：var canvasObj = canvasElem.getContext('2d') ，将上下文设为平面；
  • 通过 fillStyle 属性设置填充颜色；
  • 使用 fillRect() 方法可填充出一个矩形，4 个参数分别为（矩形左上角距画布左上角）：X 轴距离，Y 轴距离，宽度，高度；
  • 通过 strokeStyle 属性设置描边颜色；
  • 使用 strokeRect() 方法可描绘出一个矩形，参数同上；
  • 使用 clearRect() 方法可生成一个方形橡皮擦，参数同上；
  • 使用 arc() 方法可以描绘出一个圆形，6 个参数分别为（圆心距画布左上角）：X 轴距离，Y 轴距离，半径，开始弧度，结束弧度，是否为逆时针；
  • 除了画矩形，画其他图形都要以 beginPath() 开始，以 closePath() 结束；
  • 使用 lineWidth 属性可以设置线的粗细；
  • 连续画线的关键在于，起始点与结束点的交替（beginPoint = newPoint）；
